# How to run

### Clone the repo

```
git clone https://github.com/tiny-dust/docs.tinydust.cn.git
cd docs.tinydust.cn/
```

### Install dependencies using npm

```
npm install
```

or if you are in China Mainland, you'd better use `cnpm`

```
sudo npm install -g cnpm
cnpm install
```

### Here we go
```
npm start
```
